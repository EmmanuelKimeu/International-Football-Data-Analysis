International Football Results Analysis (1872–2026)

Project Overview
[cite_start]This project performs a basic exploratory data analysis (EDA) on a Kaggle dataset containing international football results from 1872 to 2024[cite: 2, 3]. [cite_start]The goal was to practice data loading, cleaning, and visualization using Python[cite: 1, 4].

Steps Performed
1. [cite_start]**Data Loading:** Imported the `results.csv` file using the Pandas library[cite: 5, 6, 7].
2. [cite_start]**Basic Exploration:** Identified the dataset size, date ranges (1872–2024), and unique participating countries[cite: 9, 10, 11, 12].
3. [cite_start]**Goals Analysis:** Calculated total goals per match and determined average scoring trends[cite: 18, 19, 22].
4. [cite_start]**Match Result Logic:** Created a custom function to classify matches as Home Wins, Away Wins, or Draws to evaluate "Home Advantage"[cite: 26, 27, 28, 37, 38].
5. [cite_start]**Visualization:** Generated histograms for goal distribution and bar charts for match outcomes and top-winning teams[cite: 40, 41, 42, 43, 44].

Technologies Used
* Python
* Pandas
* Matplotlib / Seaborn
* Jupyter Notebook
